🚀 Visual Farm Bot (Cookie Run Automation)
บอทฟาร์มอัตโนมัติที่ทำงานด้วยระบบ Computer Vision (OpenCV) ออกแบบมาเพื่อประสิทธิภาพสูงสุด กิน CPU ต่ำ และสแกนหน้าจอด้วยความเร็วสูงระดับเสี้ยววินาทีผ่านระบบ Smart Cache (Scale & Location ROI)

📋 Requirements (สิ่งที่ต้องมี)
OS: Windows (จำเป็นต้องใช้ระบบ win32api ในการควบคุมเมาส์ระดับลึกเพื่อจำลองการคลิกแบบมนุษย์)

Python: เวอร์ชัน 3.8 ขึ้นไป

Emulator: LDPlayer (หรือ Emulator อื่นๆ ที่สามารถตั้งค่าหน้าต่างให้อยู่บนหน้าจอได้)

🛠️ Installation (การติดตั้ง)
1. ติดตั้งไลบรารีที่จำเป็น
เปิด Terminal หรือ Command Prompt แล้วรันคำสั่งต่อไปนี้เพื่อติดตั้ง Dependencies ทั้งหมด:

Bash
pip install opencv-python numpy pyautogui pywin32 keyboard
2. โครงสร้างโฟลเดอร์ (Directory Structure)
โปรดตรวจสอบให้แน่ใจว่าคุณได้จัดเตรียมไฟล์รูปภาพ (.png) ไว้ในโฟลเดอร์ public/assets/ อย่างถูกต้องตามที่โค้ดอ้างอิง:

Plaintext
project-root/
│
├── farmming.py               # โค้ดหลักของบอท
├── smart_cache.json          # ไฟล์แคช (บอทจะสร้างขึ้นมาเองอัตโนมัติเมื่อรัน)
└── public/
    └── assets/
        ├── btn_main_play.png
        ├── txt_double_coin.png
        ├── btn_chest_1200.png
        ├── btn_multi.png
        ├── btn_multi_buy.png
        ├── btn_play_boost.png
        ├── btn_relay.png     # ⚠️ สำคัญ: ควรแคปรูปให้เล็กที่สุด (เช่น แค่ดวงตาตัวละคร) ไม่ติดพื้นหลัง
        ├── btn_ok.png
        ├── btn_open_all.png
        ├── btn_confirm_blue.png
        └── btn_confirm.png
🚀 How to Run (วิธีเริ่มใช้งาน)
เปิดเกมใน LDPlayer และเตรียมหน้าจอให้อยู่ใน หน้าแรก (Lobby) หรือหน้าเตรียมตัววิ่ง

รันสคริปต์ผ่าน Terminal (แนะนำให้รันด้วยสิทธิ์ Administrator เพื่อป้องกัน Emulator บล็อกคำสั่งจำลองเมาส์):

Bash
python farmming.py
การตั้งค่า Cache เริ่มต้น:
เมื่อโปรแกรมเริ่มทำงาน จะมีคำถามปรากฏขึ้นใน Terminal:

❓ คุณได้เปลี่ยนขนาดหน้าต่าง LDPlayer จากรอบที่แล้วหรือไม่? (y/n):

พิมพ์ y (Yes): หากรันครั้งแรก หรือมีการย่อ/ขยายขนาดหน้าต่างเกมใหม่ บอทจะทำการเคลียร์ข้อมูลเก่า และกวาดสายตาสร้าง Cache พิกัดใหม่ทั้งหมด

พิมพ์ n (No): หากหน้าต่างเกมมีขนาดและตำแหน่งเท่าเดิม บอทจะโหลด Smart Cache เดิมมาใช้ ทำให้ความเร็วในการประมวลผลเพิ่มขึ้นทันที

🛑 การหยุดทำงานฉุกเฉิน (Emergency Stop)
หากต้องการหยุดการทำงานของบอท ให้กดปุ่ม q บนคีย์บอร์ดค้างไว้ จนกว่าโปรแกรมจะแสดงข้อความ 🛑 หยุดทำงานฉุกเฉิน! แล้วปิดตัวเองลง

💡 Pro Tips & Troubleshooting
คลิกไม่ติด / เกมไม่ตอบสนอง: หากบอทมองเห็นปุ่มและขึ้นข้อความว่ากำลังคลิก แต่เมาส์ไม่ขยับ ให้ปิด Editor/Terminal แล้วเปิดใหม่ด้วยการคลิกขวาเลือก Run as Administrator

Self-Healing (ย้ายจอได้อิสระ): หากระหว่างที่บอททำงานอยู่ คุณจับหน้าต่าง LDPlayer ย้ายไปวางตำแหน่งอื่นบนหน้าจอ บอทจะรู้ตัวโดยอัตโนมัติว่าหาปุ่มไม่เจอ มันจะเคลียร์ Cache พิกัดตรงนั้นทิ้ง และสแกนหาหน้าต่างใหม่ให้เองทันที

ปรับแต่งความแม่นยำจังหวะวิ่งผลัด: หากบอทกดวิ่งผลัดช้าเกินไป ให้เช็คไฟล์ btn_relay.png ว่าติดพื้นหลังเกม (Dynamic Background) มาเยอะเกินไปหรือไม่ การครอปรูปให้เหลือแค่สีทึบของตัวละครจะช่วยให้บอทล็อกเป้าได้ไวที่สุด
